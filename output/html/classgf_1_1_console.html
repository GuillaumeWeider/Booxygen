Description

A virtual console.

A console is a virtual terminal where you can print the characters from a console font. Each cell of the console has a background color, a foreground color and a 8-bit character.A console has a state with default values for different aspects: a default background color (initiallly gf::Color::Black)a default foreground color (initiallly gf::Color::White)a default console effect (initiallly gf::ConsoleEffect::None)a default alignment (initiallly gf::ConsoleAlignment::Left) Several functions use this state to determine actual values.gf::ConsoleFont
