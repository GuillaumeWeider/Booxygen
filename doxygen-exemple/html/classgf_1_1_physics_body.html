Detailed Description

A physics body.

A body has several properties:

  • restitution (default: 0)
  • static and dynamic friction (default: 0)
  • linear damping (default: 0)
  • density (default: 1)
See also
gf::PhysicsGeometry, gf::PhysicsModel
